# Events

You can define events, by editing/adding files inside the [`content/forms/**`](../content/forms)

There is more understanding of what and how you can define events by reading [`nuxt.config.ts`](../nuxt.config.ts)

## Views
- [`app/pages/events/index.vue`](../app/pages/events/index.vue)
- [`app/pages/events/[...eventPath].vue`](../app/pages/events/[...eventPath].vue)
