# OAuth Server

## Prepare:

```bash
export PRIVATE_KEY=private.pem
export PUBLIC_KEY=public.pem
```

## Generate RSA key pair

### Generate the private.pem

```bash
openssl genrsa -out $PRIVATE_KEY 2048
```

### Generate the public.pem

```bash
openssl rsa -in $PRIVATE_KEY -pubout -out $PUBLIC_KEY
```

## Convert to single-line format for .env

### Convert private.pem to single-line format

```bash
openssl pkcs8 -topk8 -nocrypt -in $PRIVATE_KEY 2>/dev/null | awk '{printf "%s\\n", $0}'
```

### Convert public.pem to single-line format

```bash
awk '{printf "%s\\n", $0}' $PUBLIC_KEY
```

### Convert private.pem to public.pem single-line format

```bash
openssl rsa -in $PRIVATE_KEY -pubout 2>/dev/null | awk '{printf "%s\\n", $0}'
```
